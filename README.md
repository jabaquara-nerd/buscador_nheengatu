
# Buscador Nheengatu

Este aplicativo é um buscador do nheengatu que apresenta o termo buscado, segmentação, glosa, tradução, e contexto da palavra buscada.

## Usage/Examples

```shell
shiny run --reload --launch-browser app.py
```
